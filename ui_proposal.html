Предложения по UI

  • 1. Темная киберпанк-тема (Dark Theme)

    Избавиться от дефолтного системного дизайна. Использовать глобальный qdarktheme для Qt, добавив неоновые акценты (голубой #00bcd4 и фиолетовый #9c27b0), как видно левее. Это придаст приложению премиальный "геймерский" вид.

  • 2. Эффект матового стекла (Glassmorphism)

    Для контейнеров и карточек использовать прозрачно-черные фоны с закругленными углами и тонкими рамками (border: 1px solid rgba(255,255,255,0.1)). Это не утяжеляет интерфейс и выглядит "дорого".

  • 3. Динамичные графики в реальном времени

    Вместо текста использовать модули `pyqtgraph`. Когда сессия завершается, рисовать красивый сглаженный график движений с градиентной заливкой и радар-чарт с 5-ю осями скилла.

  • 4. Современная Типографика

    Подключить шрифт Inter или Segoe UI Variable вместо стандартного Tahoma/Arial. Огромные жирные цифры для метрик и контрастный мелкий текст для подписей.

  • 5. Микро-взаимодействия (Анимации)

    Добавить QPropertyAnimation на кнопки: плавное изменение цвета при наведении (как кнопки слева), масштабирование и красивые переходы (FadeIn) при смене вкладок или загрузке результатов ИИ.
